1) Setup: 

To run the python scripts, first execute the following
in a command line:
>pip install numpy
>pip install pandas
>pip install sklearn





2) Running scripts:

To run the main script, execute
>python main.py
from a command line in the root folder "scripts".
this will run the script with a default classifier (Gaussian Naive Bayes)
on a default dataset (TODOdefaultDatasetName).

additional command line arguments can be specified in the following way:
>python main.py <classifier> <dataset> <classifier_parameter_1> <classifier_parameter_2>...

<classifier> refers to the classifier name and should be 1 of the following 4 values:
Bayes -> for GaussianNB,
KNN -> for KNeighborsClassifier,
SVM -> for SVC,
DTree -> for DecisionTreeClassifier.
All 4 classifiers taken from the python "sklearn" library.

<dataset> refers to the dataset name (WITHOUT the .data extension) and should be 1
of the 15 dataset file names contained in the "scripts/datasets" folder
